Hello, Bootstrap 5 (Local Files)!

Purpose of each folder: css/ Contains Bootstrap's Cascading Style Sheets (CSS) files. bootstrap.css and bootstrap.min.css are the core CSS files that style all Bootstrap components (buttons, grids, typography, forms, etc.). .min.css files are minified versions (smaller file size for faster loading). bootstrap.rtl.css is for right-to-left languages support. This folder is what you link in your HTML for styling your page with Bootstrap. js/ Contains Bootstrap’s JavaScript files. bootstrap.bundle.js and bootstrap.bundle.min.js include all Bootstrap JS plugins plus the Popper.js library (required for things like tooltips, dropdowns, popovers). bootstrap.js and bootstrap.min.js are Bootstrap JS plugins without Popper.js, so you'd have to include Popper separately if you use these. These files enable interactive components like modals, carousels, dropdowns, tooltips, collapses, and so on. icons/ Contains Bootstrap Icons, an open-source icon library designed to complement Bootstrap. You get SVG files, CSS files for embedding icons, and font files (.woff, .ttf, etc.) to use the icons as fonts. This folder is optional and used only if you want to include icons in your UI.
